# 世界消失的三章

## 目录结构

```
ThreeSection/
├── assets/               # 静态资源
│   ├── images/           # 图片资源
│   ├── videos/           # 视频资源
│   ├── js/               # 脚本文件
│   └── audio/            # 音频资源
├── styles/               # CSS 样式
│   ├── normal.css        # 默认样式
│   ├── keyframes.css     # 动画关键帧定义
│   └── index/            # 页面特定样式
│       └── index.css     # 首页样式
└── index.html            # 主页面
```
